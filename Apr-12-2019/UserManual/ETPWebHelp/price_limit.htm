Price Limit

 

This report type shows the price limit for selected symbol, warehouse and year. User can only select symbol and view the price range for all warehouse and year. To do this,

  • Click on Price Limit submenu system display the report in a format shown below.

  • User is just expected to select or fill all or some of the filtering criteria à Click on Filter buttonàThe report will be displayed accordingly at the right part of the screen.

  • Symbol- Commodity symbols will be populated so that the user can select

  • Warehouse-All available warehouses will be populated

  • Year-Production year of the commodities will be displayed.
